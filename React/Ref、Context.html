    # Ref、Context

    # Refs

    允许我们访问DOM节点或者组件的实例

    • 管理焦点、文本选择或者媒体播放
    • 触发强制动画
    • 集成第三方DOM库

    # 创建ref

    class MyClass extends Component {
   construtor(){
       this.myRef = React.createRef()
   }
   render(){
       return (
           <div ref={this.myRef}></div>
       )
   }
}

    # 访问Refs

    const node = this.myRef.current

    # Refs回调

    class custom extends Component {
  constructor(){
      this.myRef = null
  }
  componentDidMount() {
    this.myRef.focus();
  }
  render(){
      return (<input ref={el => this.myRef = el} />)
  }
}
//父组件向子组件传递
class Parent extends Component {
    constructor(){
      this.childRef = null
    }
    render(){
      return (<Child childRef={el => this.myRef = el} />)
    }
}
class Child extends Component {
    render(){
      return (<input ref={props.childRef} />)
    }
}

    # Refs转发

    React.forward 接收props和refs 返回一个组件

    const FancyButton = React.forwardRef((props, ref) => (<button ref={ref}>fancy</button>))

class Parent extends Component{
    construtor(){
       this.ref = React.createRef()
    }
    render(){
        return (<FancyButton ref={this.ref}/>)
    }
}
//高阶组件
function logProps(WrappedComponent){
  class logComponent extends Component {
     render(){
       const {forwardRef} = this.props
       return (<WrappedComponent ref={forwardRef}/>)
     }
  }
  return React.forward((props, ref) => {
    return <logComponent forwardRef={ref}/>
  })
}
return logProps(FancyButton)

    # Context

    通过组件树提供了传递数据的方法，避免了手动层层传递

    # 使用

    创建一个Context对象，当渲染一个订阅了这个Context的组件，这个组件会从组件树中离自身最近的那个Provider获取Context的值,如果没有匹配到Provider，defaultValue将会生效

    const MyContext = React.createContext({
    theme: themes.light
})

    通过class.contextType挂载,

    class MyComponent extends React.Component{

}
MyComponent.contextType = MyContext

class MyClass extends Component {
  static contextType = MyContext
  render(){
      let value = this.context
  }
}
