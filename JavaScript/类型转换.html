    # 原始值转数字

    可以使用Number函数将类型转为数字类型，如果无法转换为数字，就返回NaN

    不传参数，返回+0 有参数 调用ToNumber

    参数类型 结果
    Boolean true为1，false为+0
    Null +0
    Undefined NaN
    String 前缀是0x 转为16进制，忽略前导0，如果其中有一个不是数字，则返回NaN

    # 对象转字符串或者数字

    • 当我们用String方法转换一个值的时候，如果这个值是基本类型，则参照上表，如果不是基本类型，将会调用一个ToPrimitive(input, string)方法，使其转换为基本类型，再参考上表
    • 当我们用Number方法转换一个值的时候，如果这个值是基本类型，则参照上表，如果不是基本类型，将会调用一个ToPrimitive(input, number)方法，使其转换为基本类型，再参考上表

    ToPrimitive(input, type)：当type没有指定时，input是日期，则默认string，否则默认number

    ToPrimitive(input, string)

    1. 如果input是基本类型，直接返回
    2. 调用toString() 如果是基本类型，则返回
    3. 调用valueOf() 如果是基本类型，则返回
    4. 抛错

    ToPrimitive(input, number)

    1. 如果input是基本类型，直接返回
    2. 调用valueOf() 如果是基本类型，则返回
    3. 调用toString() 如果是基本类型，则返回
    4. 抛错
    • toString
    1. 数组的toString返回一个一个字符串，元素之间添加都好
    2. 函数的toString，返回源代码
    3. 日期的toString，返回一个可读的日期
    4. 正则返回直接量的字符串
    • valueOf 返回对象本身，date会返回毫秒数

    # 一元操作符

    当+作为一元操作符，会调用ToNumber，当成Number()处理该值

    console.log(+[]);
console.log(+['1']);
console.log(+['1', '2', '3']);
console.log(+{});
console.log(+null);
console.log(+undefined);

    # 二元操作符 +

    当计算value1+value2

    1. lprim = ToPrimitive(value1)
    2. rprim = ToPrimitive(value2)
    3. lprim是字符串或者rprim是字符串，则返回toString(lprim)和toString(rprim)拼接
    4. 否则ToNumber(lprim) 和 ToNumber(rprim)的运算结果
    null+1
[] + []
[] + {}
