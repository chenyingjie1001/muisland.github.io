    # 基本类型

    # String

    • 最大字符串长度 2^53-1
    • charAt includes concat indexOf trim replace toLowerCase toUpperCase
    • slice(开始位置，结束位置) 为负数 负数+长度 开始位置<结束位置 则返回空
    • substring(开始位置，结束位置) 为负数 则为0 开始位置<结束位置 交换位置
    • substr(开始位置，长度)

    # Number

    • 2^64 - 2^53 + 3
    • NaN Infinity -Infinity
    • 双精度浮点数比较不用== 按64位存储，1位正负 11位指数，52位存储数字，0.1在转换成二进制的时候会变成无限循环小数，存在精度丢失
    • Math.abs(0.1+0.2-0.3) < Number.ESPLION()
    • 小数转二进制：乘2取整 顺序排列
    • 整数转二进制：除2取余 倒序排列
