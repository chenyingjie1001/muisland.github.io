    # 类型判断

    # 1. typeof

    • 基本类型中除了null为object,其他判断都正确
    • object中，除了function,其他都是object
    • 无法判断Date Mate RegExp等

    # 2. intanceof

    检测构造函数的prototype是否存在在某个对象的原型链上
    无法判断Date Mate RegExp、null、undefined等

    # 3. construtor

    无法判断null undefined

    var aa=[1,2];
aa.construtor === Array //true
aa.construtor === Object //true
var bb = \^$\
bb.construtor == RegExp //true
bb.construtor === Object //true
